# Express app with Supertest

Prereqs:

 - `node`
 - `yarn` | | `npm`

Usage:

#### Running tests

```sh
$ git clone https://github.com/shawnpetros/express-test.git
$ cd express-test
$ yarn && yarn test
```

#### Running the route

```sh
$ yarn start
```

In another term

```sh
$ curl localhost:3001
```
